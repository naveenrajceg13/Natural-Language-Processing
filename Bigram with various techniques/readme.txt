Readme.txt

1) Compile CalculateUniBiGram.java
2) Compile BiGram.java

javac *.java

Class Files will be generated. 

3) Run BiGram.Java with the input file name and input sentence. 

Sentence.txt contains the input sentence, to change the sentence and test. Change sentence in Sentence.txt and check.

Note: Sentence.txt should contain only one sentence, what is intented to test. For 2nd sentence file has to be changed and tested. 

java BiGram Sentence.txt input.txt
